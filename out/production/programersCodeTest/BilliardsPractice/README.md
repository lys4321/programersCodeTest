- 프로래머스 Lv2 당구 연습 문제
    - 해당 문제의 난이도 상승 원인
        - 처음엔  '같은 입사각' 이란 설명에 반응하여 삼각형 구조를 떠올려서 피타고라스 공식을 사용해 문제를 해결하려 했으나, 그로 인하여 불필요한 생각과 처리에 어려움이 발생해 해결에 어려움을 느낌
    - 해당 문제를 해결한 방법
        - 이 문제를 해결하는 방법은 '같은 입사각'을 이해하는 방식에 따라 갈린 것 같음 처음의 나는 삼각형-피타고라스 공식에 집중을 하였지만  사실 해결법은 이 의미를 사용하는 다른 방법에 있었음 그것은 축을 기준으로 대칭상태를 만들어 단순히 좌표와 좌표간 길이를 비교하는 방법이 었음 물론 그 중에서도 '벽을 튕겨야 한다'라는 제한이 있어 같은 축 상에 있을 때 예외처리를 해야한다는 것도 필요하였음
    -  처음 방식에서 변경한 방식
        - 처음
            - 튕겨져 나온 입사각에 따른 이동된 좌표를 계산해 목표까지의 길이를 피타고라스 공식을 사용해 계산
        - 변경
            - 4가지의 축을 기준으로 좌표를 대칭시켜 4가지의 직선형태로 만들고 그 중에서도 벽을 튕기지 않는 경우를 제외한 후 제일 짧은 길이의 값을 고름
    - 문제의 핵심
        - 축을 사용하여 각이 아닌 직선을 사용해 길이 계산
    - 이 문제의 목적
        - 보여지는 좌표 그 자체를 사용하는 것이 아닌 이를 추가적으로 사용해 사용하는 것
