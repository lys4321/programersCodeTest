- 프로래머스 Lv2 완전범죄 문제
    - 해당 문제의 난이도 상승 원인
        - DP 알고리즘에 대하 이해 필요
        - INF 같은 무식하게 큰 값을 사용한 비교를 사용한 방법
    - 해당 문제를 해결한 방법
        - 기존에 해결했던 문제를 참고 및 추가적인 알고리즘에 대한 검색을 통해 이해 강화를 통하여 문제를 해결함
    -  처음 방식에서 변경한 방식
        - 처음
            - 이전에 사용했었던 재귀함수를 이용한 방식으로 문제를 해결하려 하였으나 작업할 당시에는 이 문제가 그 때의 문제처럼 해결할 수 없다는 것을 인지 못하여 과부하 및 오류가 발생
        - 변경
            - 재귀함수 방식에서 변경하여 반복문을 통해 한번의 반복이 완료될 시 나온 최솟값들을 따로 저장한 후에 결과 도출 시 모든 최솟값들의 최솟값을 찾아 정답으로 사용
    - 문제의 핵심
        - A도둑이 최소로 물건을 훔치고 B도둑이 나머지를 훔칠 수 있는지가 중요
    - 이 문제의 목적
        - DP, BFS, DFS 등의 알고리즘에 대한 이해 및 사용해야할 때가 있다는 것을 일깨움
        - DP란 무엇인가를 학습
